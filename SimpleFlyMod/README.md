# Simple Fly Mod
Just A Simple Fly Mod / Collision mod

## Required mods
- [HydroUMH](https://github.com/RHlNO/HydroneerModding/raw/main/Release%20Mods/501-HydroUMH_P.pak)
-----------
## Keybinds
- `F` : Toggle fly
- `F2` : Toggle the UI
- `G` : Toggle Collision
- `Shift + G` : Toggle World Collision
- `space` : Fly up
- `c` : Fly down
